Improving Cystatin C Testing in in Patients Undergoing Cardiac Catheterization

A quality improvement project focused on increasing cystatin C testing among inpatients undergoing cardiac catheterization to improve the accuracy of kidney function assessment.

Overview Problem Goal & Specific Aims Methods Team Contact

Project Overview

Cystatin C testing is recommended by the Kidney Disease Improving Global Outcomes (KDIGO) when accurate assessment of kidney function is needed, or when creatinine-based eGFR (eGFRcr) may be unreliable.1 This quality improvement (QI) initiative was designed to increase health provider familiarity and adherence with cystatin C testing to improve the accuracy of kidney function assessment, and ultimately to improve patient care.

Problem Statement & Background

About 40% of patients undergoing cardiac catheterization have concomitant CKD. To classify CKD, GFR estimation using a combined creatine and cystatin C equation (eGFRcr-cys) provides the best accuracy.2 However, cystatin C is underutilized.

Goal & Specific Aims

Goal

The overarching goal is to improve cystatin C testing in high-risk patients..

Specific Aims

  • Increasing ordering of cystatin C.
  • Enhance health provider knowledge and confidence related to cystatin C testing.
  • Evaluate the discrenpancy between creatinine-based eGFR (eGFRcr) and cystatin C-based eGFR(eGFRcys).

Methods

QI Framework

The project is planned for one month (estimated ~160 patients). Preliminary results will contribute to the decision makeing on whether cystatin C should be included as part of the pre-catheterization protocol.

Measures

  • Process: Examination of serum creatinine and cystatin C before cardiac catheterization.
